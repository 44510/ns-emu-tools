密钥管理

由于版权原因, 本项目不再提供 keys 自动安装功能。

请用户自行放置 keys 至相应文件夹。

获取 keys

你可以跟随 yuzu 的官方教程 从 switch 中提取 keys 文件.

也可以在贴吧或者 qq 群等地方找到其他大佬分享的 keys 文件.

下面是两位大佬分享的 keys:

1. 由 文无 大佬提供的 城通网盘(访问密码：7257), 原帖地址: 百度贴吧

2. 由 PrincessAkira 大佬提供的 GDrive(需代理) 原地址: GitHub

添加 keys 到模拟器

请将上面获取到的 prod.keys 和 title.keys 放进相应模拟器的文件夹

注意: 你使用的 keys 版本号应该大于等于当前使用的固件版本号
